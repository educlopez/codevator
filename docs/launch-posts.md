### LinkedIn (EN)

```
I've been coding with Claude Code a lot lately.

And I noticed something: when the agent is working, there's this silence. You're just... waiting. Staring at the screen. You don't know if it's reading, editing, or having an existential crisis about your variable names.

It felt like waiting inside a Lumon elevator. So I leaned into it.

I built Codevator — a CLI that plays elevator music while your AI coding agent works. It stops when it needs your attention.

One command:
npx codevator

It hooks directly into Claude Code. No config files, no setup wizards. Music starts when the agent works, stops when it finishes or needs your input.

5 modes: elevator, typewriter, ambient, retro, and minimal. Each one carefully selected for maximum productivity (and mild existential comfort).

The website has a Severance-inspired elevator you have to "open" before seeing anything. There are draggable post-it notes. The testimonials are from fictional employees of Lumon Industries and Dunder Mifflin.

Because sometimes the tools we build don't have to be serious. Sometimes they just have to make the wait a little less weird.

Open source (MIT). Try it and let me know which mode you work best with.

codevator.dev
```

### Day 6 — Technical detail

**X.com (EN):**

```
How codevator works (your outie may find this interesting):

1. npx codevator → writes hooks to ~/.claude/settings.json
2. PreToolUse → music starts
3. Stop → music stops
4. Notification → music stops on permission prompts

Native audio player. No Electron. No browser. No waffle party required.

The code is open. The severed floor is not.

github.com/educlopez/codevator
```

**X.com (ES):**

```
Cómo funciona codevator (a tu outie le puede interesar):

1. npx codevator → escribe hooks en ~/.claude/settings.json
2. PreToolUse → la música empieza
3. Stop → la música para
4. Notification → para en prompts de permiso

Reproductor nativo. Sin Electron. Sin browser. Sin waffle party necesaria.

El código es abierto. La planta cortada no.

github.com/educlopez/codevator
```
